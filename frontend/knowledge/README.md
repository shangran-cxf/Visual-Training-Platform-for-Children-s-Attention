# 注意力科普知识内容管理指南

## 文件结构说明

### `/frontend/knowledge/` 目录结构
```
knowledge/
├── knowledge-data.json          # 主要知识内容数据文件
├── templates/                   # 内容模板目录
│   ├── article-template.json    # 文章数据模板
│   └── category-template.md     # 分类说明模板
└── archives/                    # 历史内容归档目录
```

## 如何添加新知识内容

### 方法一：编辑 JSON 文件
直接编辑 `knowledge-data.json` 文件，按照以下格式添加新文章：

```json
{
  "id": 9,
  "title": "新文章标题",
  "content": "这里是文章的详细内容...",
  "author": "作者名称",
  "date": "YYYY-MM-DD",
  "category": "分类名称",
  "tags": ["标签1", "标签2", "标签3"]
}
```

### 方法二：使用内容管理系统
你可以创建一个简单的管理界面来添加和编辑内容。

## 内容分类建议

- **发展心理学**：关于儿童注意力发展的理论研究
- **训练方法**：具体的注意力训练技巧和方法
- **家庭教育**：家庭环境对孩子注意力的影响
- **注意力类型**：选择性注意、持续性注意等不同类型介绍
- **感知训练**：视觉、听觉等感知能力训练
- **认知能力**：工作记忆、执行功能等相关认知能力
- **自控能力**：抑制控制、情绪调节等内容

## 内容质量标准

- **准确性**：确保所有信息基于科学研究或权威来源
- **实用性**：提供家长可以实际操作的建议
- **易懂性**：使用通俗易懂的语言解释专业概念
- **相关性**：内容应与注意力训练直接相关
